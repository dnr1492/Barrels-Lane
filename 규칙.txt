***게임 플레이 규칙

*덱
- 랭크 상관 없이 이름 기준으로 동일한 스킬카드 최대 4장
- 캐릭터 수 * 4장 == 덱 수

*패
!턴 시작 시
- 캐릭터 수만큼 기본 이동카드(스킬카드와 별개) 보유
- 캐릭터 수 * 2만큼 뽑기

!턴 진행 시
- 총 4장을 뒷면으로 셋팅 
         (단, 제한시간 초과 등으로 셋팅을 못하더라도 그대로 진행)
  *캐릭터 사망 시 셋팅해 둔 해당 스킬 카드는 발동해도 아무런 효과가 없음

!턴 종료 시
- 패에 나머지 카드 전부 덱에 회수 후 랜덤 셔플
         (단, 캐릭터 수만큼 기본 이동카드(스킬카드와 별개)는 패에 반환)
- 캐릭터 사망 시 덱에서 해당 캐릭터의 스킬카드 전부와 기본 이동카드(스킬카드와 별개) 1장 제거

*기타 룰
- 동전으로 앞/뒷면 선택 및 선공권
- 선공부터 턴마다 지그재그 형식으로 4Round로 진행 
- Leader 사망 시 패배
- 양쪽 Leader만 남고 모든 캐릭터 사망한 상태인 경우 플레이어한테 무승부 결정 여부를 제공
- (덱) 캐릭터 선택
            maxCost = 10,
            maxLeaderCount = 1,
            maxHighTierCount = 2,
            maxMiddleTierCount = 3,
            maxLowTierCount = 5,
            limitMiddle = 2,
            limitLow = 3,
